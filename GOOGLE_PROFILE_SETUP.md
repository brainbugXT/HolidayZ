## ðŸ”§ Developer Note

The avatar utility (`src/utils/avatar.ts`) now uses:
- **Proper MD5 hashing** (via `md5` npm package)
- **Gravatar API** with `d=retro` fallback
- **Consistent colors** for initials fallback

### Gravatar URL Structure:
```
https://www.gravatar.com/avatar/MD5_HASH?s=SIZE&d=retro
```

- `MD5_HASH`: MD5 of lowercase, trimmed email
- `SIZE`: Image size in pixels (64px for app)
- `d=retro`: Fallback to 8-bit style avatar if no Gravatar

## ðŸŽ¨ Fallback Options (Can Change)

In `src/utils/avatar.ts`, you can change the `d=` parameter:

```typescript
// Current: Retro 8-bit style
d=retro

// Other options:
d=mp          // Mystery person (gray silhouette)
d=identicon   // Geometric pattern
d=monsterid   // Monster avatar
d=robohash    // Robot avatar
d=wavatar     // Generated face
```

To change, edit line in `avatar.ts`:
```typescript
return `https://www.gravatar.com/avatar/${hash}?s=${size}&d=retro`;
//                                                           ^^^^^ Change here
```
